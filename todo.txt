iOS CP/M Emulator - Status
===========================

COMPLETED (Dec 2024):

1. Replaced duplicate sources with symlinks to sister directories
   - qkz80 files now symlink to ../cpmemu/src/
   - hbios_dispatch, emu_io.h, romwbw_mem.h symlink to ../romwbw_emu/src/
   - Kept iOS-specific: emu_io_ios.mm, hbios_core.cc/h

2. Added critical HBIOS fixes from IOS_PORTING.md
   - APITYPE patching at ROM offset 0x0112 (set to 0x00 for HBIOS)
   - HBIOS ident signatures at 0xFF00 and 0xFE00 in common area
   - SYSRESET callback for REBOOT command support

3. Added ROM/Disk selection UI (matching web version)
   - 4 ROM options: SBC_simh_std, emu_romwbw, RCZ80_std, emu_rcz80
   - 4 Disk slots with bundled options: CP/M 2.2, ZSDOS, QPM, Drive A
   - Boot string text field for auto-boot commands
   - Settings sheet accessible via gear icon

4. Fixed font size not working
   - Removed terminal scaling - font size now directly affects text size
   - Larger fonts = larger text, smaller fonts = smaller text

5. Added debug toggle in Settings
   - Reduces console spam when disabled (off by default)
   - Removed TerminalView debug print statements

6. Added local disk file support
   - Open existing .img files from device
   - Create new 8MB disk images
   - Save changes back to file-backed disks
   - 8MB max size (8-bit OS addressing limit)

SISTER PROJECTS:
- ../cpmemu - Z80 CPU emulator (qkz80)
- ../romwbw_emu - RomWBW HBIOS emulation
  - romwbw_emu/web - WebAssembly version (reference implementation)
  - romwbw_emu/docs/IOS_PORTING.md - porting notes

BUILD:
- Builds successfully for Mac Catalyst
- Code signing may need keychain access for device builds
